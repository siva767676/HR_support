Overview

About AI Resume Screening

An enterprise-grade screening assistant that ranks large batches of resumes against a job description using a two-stage AI pipeline — fast semantic shortlisting followed by a detailed LLM evaluation — so recruiters spend their time on the strongest candidates.

Two-Stage Pipeline

Embedding-based similarity shortlists the most relevant resumes, then an LLM scores each one in depth.

Weighted Scoring

Skills, experience, education, domain relevance and projects combine into one transparent 0–100 score.

Explainable Results

Every candidate comes with matched skills, gaps, strengths and a written recommendation.

Private by Design

Nothing is persisted — each screening is a standalone analysis run entirely on your infrastructure.

How it works

  1. Open Get Started and select a folder of resumes plus one or more JD files.
  2. Click Analyze — resumes are extracted, embedded and shortlisted by similarity.
  3. The shortlist is evaluated by the LLM and ranked by overall match score.
  4. Review ranked results, drill into Shortlisted Candidates, and track stats on the Dashboard.
